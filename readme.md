## Cómo Empezar (Guía para Contribuidores/Interesados)

###  Inicio Rápido (Local)
Para obtener una copia local y funcionando, sigan estos sencillos pasos:

Clonar el repositorio:

- Paso 1 Bash
git clone https://github.com/SuUsuario/dream-developers.git
Moverse al directorio del proyecto. 
- Paso 2 Bash
cd dream-developers.

- Paso 3 Abrir con VS Code: Bash
code .

- Paso 4 Ejecutar con Live Server:
     Asegúrense de tener instalada la extensión "Live Server" de Ritwick Dey en VS Code.
     Hagan clic derecho sobre el archivo index.html (o el principal) y seleccionen "Open with Live Server". El proyecto se abrirá automáticamente en su navegador en http://127.0.0.1:5500/.

###  Contribuciones

> Estamos emocionados de que quieran ayudarnos a mejorar **Dream Developers**! Aceptamos cualquier sugerencia, corrección de errores o nueva funcionalidad. La colaboración enriquece nuestro aprendizaje.

Para contribuir, por favor, sigan el flujo de trabajo estándar de GitHub:

1.  **Hagan un *Fork***: Creen una copia del repositorio en su cuenta de GitHub.
2.  **Clonen su *Fork***: Bajen la copia a su máquina local.
3.  **Creen una Rama de Trabajo:** Aíslen sus cambios creando una rama específica.
    ```bash
    git checkout -b feature/nombre-de-su-cambio
    ```
4.  **Hagan *Commit*:** Guarden sus modificaciones con un mensaje claro y conciso.
    ```bash
    git commit -m 'feat: Añadir sección de contacto en el footer'
    ```
5.  **Hagan *Push*:** Suban los cambios a su repositorio en GitHub.
    ```bash
    git push origin feature/nombre-de-su-cambio
    ```
6.  **Abran un *Pull Request* (PR):** Vayan a su repositorio en GitHub y soliciten que sus cambios sean incorporados al repositorio original de Dream Developers.

Gracias por su interés y esfuerzo!
